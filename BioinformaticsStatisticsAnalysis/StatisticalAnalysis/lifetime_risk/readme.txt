## Nicholas Boddicker, PhD
## Lifetime risk of breast cancer
## December 11, 2023
## Supporting analyses documents for manuscript entitled "Saturation genome editing-based functional evaluation and clinical classification of BRCA2 single nucleotide variants"

This folder contains the following:

1) SEER21_NHW_BC_Rates.csv -- Input file that contains age, age-specific breast cancer rates, and age-specific death rates based on SEER21
2) BRCA2_MAVE_LifetimeRisk_Code.txt -- Code for calculating lifetime breast cancer risk curves and lifetime risk estimates using odds ratios calculated from gnomAD controls and Ambry Genetics breast cancer cases.
